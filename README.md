# ShrinkGPT.github.io
AI Shrink chat bot with GPT
 

# Virtual Environment
! Use python 3.9.7
1) create a virtual environment
    * python -m virtualenv venv
2) Activate the environment
    * venv\Scripts\activate
3) To deactivate
    * deactivate

# Install Environment
python -m pip install --upgrade pip
python -m pip install -r requirements.txt
